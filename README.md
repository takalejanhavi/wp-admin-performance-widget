# wp-admin-performance-widget
## Admin Performance Widget

A WordPress plugin that adds a custom admin dashboard widget
displaying basic system and WordPress information.

## Purpose
Built to understand WordPress admin hooks and clean plugin structure.

## Features
- Uses WordPress actions
- Sanitized output
- Simple and maintainable code

## Hooks Used
- wp_dashboard_setup

## Installation
1. Upload to wp-content/plugins
2. Activate from WordPress admin
